## ğŸ” Overview

EoH-S (Evolution of Heuristic Set) is a novel framework that leverages Large Language Models (LLMs) for automated heuristic design. Our approach evolves a set of diverse and effective heuristics through iterative refinement, enabling superior performance across various combinatorial optimization problems.

### Key Features

- ğŸš€ **Automated Heuristic Design**: Leverages LLMs to automatically generate and evolve heuristics
- ğŸ¯ **Set-based Evolution**: Maintains and evolves a diverse set of heuristics rather than a single solution
- ğŸ”„ **Iterative Refinement**: Continuously improves heuristic quality through evolutionary strategies
- ğŸ“Š **Broad Applicability**: Tested on multiple combinatorial optimization tasks
- âš¡ **State-of-the-art Performance**: Achieves competitive results compared to existing methods

## ğŸ› ï¸ Installation

### Prerequisites

- Python 3.8 or higher
- pip package manager
- (Optional) CUDA-capable GPU for faster execution

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/eohs.git
   cd eohs
   ```

2. **Install LLM4AD platform**
   ```bash
   cd code
   pip install .
   cd ..
   ```

3. **Install additional dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Set up API keys** (if using commercial LLM APIs)
   ```bash
   export OPENAI_API_KEY="your-api-key-here"
   # or create a .env file with your API keys
   ```

## ğŸš€ Quick Start

### Running EoH-S on Online Bin Packing

```bash
cd examples
python obp_set/run_eohs.py
```

The results will be saved in the `logs/` folder.

### Running Other Tasks

```bash
# For TSP
python tsp/run_eohs.py

# For other tasks
python <task_name>/run_eohs.py
```

### Customizing Experiments

You can modify the configuration files in `examples/` to customize:
- Population size
- Number of iterations
- LLM model selection
- Task-specific parameters

Example configuration:
```python
config = {
    "population_size": 10,
    "num_iterations": 50,
    "llm_model": "gpt-4",
    "task": "online_bin_packing"
}
```
